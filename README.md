# Zeraboard

Zeraboard is a modern web application built using React.js.  
It provides a fast and responsive user experience with modern frontend technologies.

## 🚀 Features

- Modern React.js UI
- Responsive design with Tailwind CSS
- Global state management using Context API
- State logic handled using Reducer Hook
- Firebase integration
- API data fetching using Fetch API
- Custom CSS styling
- Reusable components
- Fast and optimized performance

## 🛠️ Technologies Used

- React.js
- JavaScript (ES6+)
- Tailwind CSS
- Firebase
- Fetch API
- Context API
- Reducer Hook
- Custom CSS
- HTML5
